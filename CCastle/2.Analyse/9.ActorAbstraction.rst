================================
Actor Abstractions (Busy, DRAFT)
================================

.. post::
   :category: Castle DesignStudy
   :tags: Castle, Concurrency

   The Actor-Model_ which is inspired by physics, unlike older models like the famous Petri-net_ that are mostly
   mathematical. And has become popular for Distributed-Computing_, where each node/computing act as in independent,
   active actor.

   As an abstraction, this is similar to the “Many Core” :ref:`CC` we use for CCastle. Hence, we study this model a bit  more
